🎨 2. 색상 시스템

2.1 브랜드 색상

2.2 색상 사용 가이드라인

  • Primary: 주요 액션, 브랜드 요소
  • Secondary: 보조 액션, 부가 정보
  • Success: 성공 메시지, 승인 액션
  • Warning: 주의 메시지, 편집 액션
  • Danger: 오류 메시지, 삭제/거부 액션
  • Info: 정보 메시지, 조회 액션

⚙️ 개발 가이드라인

4.1 스타일 사용 원칙

✅ DO - 올바른 사용법
  • Bootstrap 5.2.3 표준 클래스만 사용: btn-primary, btn-outline-success
  • 의미에 맞는 색상 사용: 성공=Success, 오류=Danger, 편집=Warning
  • 일관된 간격 사용: me-2, mb-3, p-3
  • 액션에 맞는 버튼 사용: 조회=Info, 수정=Warning, 삭제=Danger
❌ DON'T - 피해야 할 사용법
  • BEM 형식 사용 금지: btn--primary ❌
  • 인라인 스타일 사용 금지: style="color: red" ❌
  • 의미 없는 색상 사용: 삭제 버튼에 Success 색상 ❌
  • 사용자 정의 CSS 클래스 생성 최소화
