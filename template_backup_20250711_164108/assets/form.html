  • 관리 IP 주소
  • 포트 수 및 PoE 포트 수
  • 업링크 속도
사무 기기 등록 가이드

  • 주요 기능 (인쇄, 복사, 스캔, 팩스)
  • 연결 방식 (유선 LAN, Wi-Fi, USB)
  • 인쇄 속도
  • 토너/잉크 모델명
  • 유지보수 주기
서버 장비 등록 가이드
필수 정보:
  • 서버 유형 (웹, 데이터베이스, 파일, 애플리케이션)
  • CPU 코어 수
  • 메모리 용량 (ECC 메모리 여부)
  • 저장장치 구성 (RAID 설정)
  • 서버 운영체제

  • 주사율 (예: 144Hz)
  • 연결 포트 (HDMI, DisplayPort, USB-C)
